HTML কি?‎ ‎
‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎
HTML এর পূর্ণরূপ HyperText Markup Language। এটি ওয়েবপেজের গঠন (Structure) তৈরি করতে ব্যবহৃত হয়। HTML দিয়ে আমরা লিখি শিরোনাম, অনুচ্ছেদ, ছবি, লিংক, টেবিল, ফর্ম ইত্যাদি।

HTML-এর উদ্বোধন: HTML প্রথম তৈরি করেছিলেন টিম বার্নার্স-লি ১৯৯১ সালে। প্রথম দিককার HTML খুব সাধারণ ছিল এবং মাত্র কিছু ট্যাগ ব্যবহার করা হতো।

HTML-এর সংস্করণ পরিবর্তন: HTML সময়ের সাথে সাথে বিভিন্ন সংস্করণে পরিবর্তিত হয়েছে। যেমন:
1.HTML 1.0 (১৯৯১) ।
2.HTML 2.0 (১৯৯৫)
3.HTML 3.2 (১৯৯৭)
4.HTML 4.01 (১৯৯৯)
5.XHTML (২০০০), এবং আধুনিক HTML5 (২০১৪)।

সারসংক্ষেপ: HTML হলো ওয়েবপেজের হাড়-গোড় (skeleton)। CSS ব্যবহার করে ডিজাইন করা হয় আর JavaScript ব্যবহার করে ওয়েবপেজে প্রাণ দেওয়া হয়।
