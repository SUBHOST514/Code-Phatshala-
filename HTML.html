HTML কি?‎ ‎
‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎
HTML এর পূর্ণরূপ HyperText Markup Language। এটি ওয়েবপেজের গঠন (Structure) তৈরি করতে ব্যবহৃত হয়। HTML দিয়ে আমরা লিখি শিরোনাম, অনুচ্ছেদ, ছবি, লিংক, টেবিল, ফর্ম ইত্যাদি।

HTML-এর উদ্বোধন: HTML প্রথম তৈরি করেছিলেন টিম বার্নার্স-লি ১৯৯১ সালে। প্রথম দিককার HTML খুব সাধারণ ছিল এবং মাত্র কিছু ট্যাগ ব্যবহার করা হতো।

HTML-এর সংস্করণ পরিবর্তন: HTML সময়ের সাথে সাথে বিভিন্ন সংস্করণে পরিবর্তিত হয়েছে। যেমন:
1.HTML 1.0 (১৯৯১) ।
2.HTML 2.0 (১৯৯৫)
3.HTML 3.2 (১৯৯৭)
4.HTML 4.01 (১৯৯৯)
5.XHTML (২০০০), এবং আধুনিক HTML5 (২০১৪)।

সারসংক্ষেপ: HTML হলো ওয়েবপেজের হাড়-গোড় (skeleton)। CSS ব্যবহার করে ডিজাইন করা হয় আর JavaScript ব্যবহার করে ওয়েবপেজে প্রাণ দেওয়া হয়।

HTML এর গল্প

1.HTML পুরো কাহিনী পড়ুন

একসময় মানুষজন শুধু বই, পত্রিকা বা গবেষণা পেপারের মাধ্যমে জ্ঞান বিনিময় করত। ১৯৯০ সালে টিম বার্নার্স-লি নামের একজন বিজ্ঞানী ভাবলেন, যদি ইন্টারনেট ব্যবহার করে তথ্য একে অপরের সাথে সহজে ভাগ করা যেত, তবে বিজ্ঞান আর প্রযুক্তি অনেক দ্রুত এগোত।

এই চিন্তা থেকেই জন্ম নিল HTML (HyperText Markup Language)। প্রথম দিকের HTML খুব সাধারণ ছিল—কিছু শিরোনাম, অনুচ্ছেদ আর লিঙ্ক দেখানোর মতো মাত্র কয়েকটা ট্যাগ ছিল। তবুও এটি ছিল বিপ্লবী আবিষ্কার, কারণ এর মাধ্যমে এক দেশ থেকে আরেক দেশে, এক বিশ্ববিদ্যালয় থেকে আরেক বিশ্ববিদ্যালয়ে জ্ঞান ছড়িয়ে পড়তে লাগল।

সময়ের সাথে সাথে HTML অনেক রূপ বদলেছে। ১৯৯৫ সালে এলো HTML 2.0, ১৯৯৭ সালে HTML 3.2, ১৯৯৯ সালে HTML 4.01, আর অবশেষে ২০১৪ সালে আধুনিক HTML5 ঘোষণা করা হলো। এখন এটি শুধু লেখা বা ছবি নয়, ভিডিও, অডিও, অ্যানিমেশন এমনকি গেমস তৈরিতেও ব্যবহৃত হচ্ছে।

আজ HTML ছাড়া ওয়েবসাইট কল্পনা করাই যায় না। যেভাবে মানুষের ভাষা আমাদের যোগাযোগের মাধ্যম, তেমনি HTML হলো ইন্টারনেটের ভাষা।

2.HTML এর গল্প

একসময় মানুষজন শুধু বই, পত্রিকা বা গবেষণা পেপারের মাধ্যমে জ্ঞান বিনিময় করত। ১৯৯০ সালে টিম বার্নার্স-লি নামের একজন বিজ্ঞানী ভাবলেন, যদি ইন্টারনেট ব্যবহার করে তথ্য একে অপরের সাথে সহজে ভাগ করা যেত, তবে বিজ্ঞান আর প্রযুক্তি অনেক দ্রুত এগোত।

এই চিন্তা থেকেই জন্ম নিল HTML। প্রথম দিকের HTML খুব সাধারণ ছিল—কিছু শিরোনাম, অনুচ্ছেদ আর লিঙ্ক দেখানোর মতো মাত্র কয়েকটা ট্যাগ ছিল। কিন্তু সেটিই আজকের ওয়েব দুনিয়ার ভিত্তি।

HTML সম্পর্কিত ঘটনা

HTML এর চেয়ারম্যান ও সংগঠন

HTML কে একক কোনো ব্যক্তির হাতে ছেড়ে দেওয়া হয়নি। ইন্টারনেট যাতে সবার জন্য সমানভাবে কাজ করে, তাই World Wide Web Consortium (W3C) গঠন করা হয়। এই সংগঠনের চেয়ারম্যান ছিলেন HTML এর উদ্ভাবক টিম বার্নার্স-লি।

পরে আরও অনেক সংগঠন HTML উন্নয়নের কাজে যোগ দেয়, যেমন WHATWG (Web Hypertext Application Technology Working Group)। তারা একসাথে কাজ করে আধুনিক HTML5 তৈরি করে, যেখানে ভিডিও, অডিও, ক্যানভাসসহ শত শত নতুন ফিচার যোগ হয়েছে।

আজও W3C এবং WHATWG এর যৌথ প্রচেষ্টাতেই HTML এর মান (Standards) তৈরি হয়। অর্থাৎ, HTML এর পেছনে শুধু একজন মানুষ নয়, বরং একটি বৈশ্বিক সংগঠন কাজ করছে।
